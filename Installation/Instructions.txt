DICOM_Pipeline environment installation

Download the miniconda executable (https://docs.conda.io/en/latest/miniconda.html) to the following location:
%USERPROFILE%\Downloads (i.e. "C:\Users\Adarsh\Downloads")

Run the miniconda executable and open a "miniconda3" prompt

cd to the _venvs folder of this repo ("DICOMAnonymizationPipeline\_venvs")

IF FIRST TIME:
    Type "conda env create --file DICOM_Env.yml" to create your environment for the first time

IF JUST PULLED:
    Type "conda env update --file DICOM_Env.yml" to update local environment from file

Type "conda activate DICOM_Env.yml" to activate virtual environment
